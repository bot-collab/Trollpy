Para poder usarlo solo tienes que instalar las siguientes librerias:
<ol>
<li>CV2</li>
<li>CustomTKinter</li>
<li>threaded</li>
<li>numpy</li>
<li>pygame</li>
</ol

<pre><code>pip install threaded
